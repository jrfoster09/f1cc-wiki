2

🔑 One-Time Setup

You only need to do this once. It lets the workflow commit the updated JSON files back to your repo automatically.

1
Go to GitHub → Settings → Developer settings → Fine-grained tokens
2
Click Generate new token. Set expiry to 1 year. Select your wiki repo only.
3
Under Permissions → Repository, set Actions → Read & Write and Contents → Read & Write.
4
In your repo go to Settings → Secrets and variables → Actions → New repository secret. Name it PAT_TOKEN, paste the token. Done! ✓
That's all you need.
No token to paste each session. The sync runs automatically every hour, and the manual trigger above requires no extra credentials.

🏁 Adding an F1 Race Result (Weekly)
1
Fill in the Race Information section — name, circuit, date, VOD link and a short summary.
2
Add each finisher using + Add Driver. Select driver, team, qualifying position. Check FL for the fastest lap setter.
3
Copy the JSON from the output panel on the right.
4
On GitHub, open data/races_YYYY.json. Find the matching race entry (by round number) and replace the entire object with the copied JSON.
5
Commit the change on GitHub. The site updates within 30 seconds.
6
Then go to the Update Standings tab and update the season standings in db.json too.
🏆 Adding an F2 Race Result
1
Switch to the Add F2 Result tab. Only F2 drivers and F2 teams appear in the dropdowns.
2
Fill in race info and add all finishers. Points are 25-18-15-12-10-8-6-4-2-1 (no FL bonus).
3
Copy the JSON and open data/races_YYYY_f2.json on GitHub. Find the round and replace the entire object.
4
Commit, then use the Update Standings tab — select the F2 season (e.g. 2026 F2) to generate updated driver_standings for db.json.
➕ Starting a New Season
1
In db.json, add a new entry inside "seasons" with status: "live" and empty standings arrays.
2
Update "currentSeason" in db.json → meta to the new year.
3
Create data/races_YYYY.json (copy the template from README) with all planned rounds set to status: "upcoming".
4
For new drivers, add them to db.json → drivers with their season entry. The site picks them up everywhere automatically.
